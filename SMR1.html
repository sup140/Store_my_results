ABOUT

 This is a Website on Student Result Management using PHP and by using this system we can manage Student result online. With this system we not only manage student result but also we can manage individual students data by Semester. The main purpose of developing this Student Result Management System in PHP is to reduce the manual errors and convert result system into computerized system.

 An Online Student Result Management System is a very well organized and appropriate system to utilize. This Student Result Management System is not only for student user but also assist the management which proceed with a large amount of pain to publish the Student results in usual conditions.
